rom-rails

Rails integration for Ruby Object Mapper which ships with:

  • Relation generators
  • Mapper generators
  • Command generators
  • Repository generators

Tests

To run tests:

RAILS_ENV=test bundle exec rake db:migrate
bundle exec rake
