rom-yesql

Yesql-like adapter for Ruby Object Mapper.

  • API docs
  • ROM user docs

Installation

Add this line to your application's Gemfile:

gem 'rom-yesql'

And then execute:

$ bundle

Or install it yourself as:

$ gem install rom-yesql
