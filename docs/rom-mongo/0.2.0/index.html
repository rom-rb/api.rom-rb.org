ROM::Mongo

MongoDB support for Ruby Object Mapper

Installation

Add this line to your application's Gemfile:

gem 'rom-mongo'

And then execute:

$ bundle

Or install it yourself as:

$ gem install rom-mongo

Usage

See spec/integration/gateway_spec.rb for a sample usage.
