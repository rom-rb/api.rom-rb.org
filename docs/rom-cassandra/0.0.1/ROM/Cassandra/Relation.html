Overview

Relation subclass of Cassandra adapter

Examples:

class Users < ROM::Relation[:cassandra]
  def last_ten_admins
    select(:id, :name)
      .where(role: "admin")
      .using(consistency: :quorum)
      .order(:name, :desc)
      .limit(10)
  end
end
